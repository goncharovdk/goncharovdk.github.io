Natural exponential e^x

The goal is to find a fixed point of differentiation operator D, i.e. find a function f(x) such that Df(x) = f'(x) = \frac{df}{dx} = f(x). This also mean that df = f(x)dx and f(x) = \int f(x). (We want integration constant C=0.) Then, fixed point implies f(x) = f'(x) = f''(x) = f^{(n)}(x).

Linear function y(x)=ax cannot be a fixed point as it is a variate and its derivative is a constant. Let’s experiment with more complex function y(x) = x^n \Rightarrow y'(x) = nx^{n-1}. It’s better but let’s try to improve it: y(x)={x^n \over n} \Rightarrow y'(x) = x^{n-1}. Not good enough yet.

Try again: y(x) = {x^n \over n!} \Rightarrow y'(x) = {x^{n-1} \over (n-1)!}. Now we can see y(x) as a parametrized function y_n(x)={x^n \over n!} so y'_n(x)=y_{n-1}(x).

At this step, note that {x^0 \equiv 1}, {0! \equiv 1} \Rightarrow y_0(x)=1 for any x. For n > 0, as 0^n \equiv 0 \Rightarrow y_n(0) = 0. Then, [y_n(ax)]' = a y'_n(ax) = a y_{n-1}(ax).

This is not enough for a fixed point as the function parameters of left-hand and right-hand sides are different. But infinite sum of such functions would do as \lim_{n \to \infty} (n) = \lim_{n \to \infty} (n-1). So a fixed point is f(x) = \sum_{n=0}^\infty {y_n(x)} = \sum_{n=0}^\infty {x^n \over n!} Regarding f(x), note that f(0) = y_0(0) = 1. Then due properties of y(x) and linearity of summation [f(ax)]' = af'(ax) = af(ax). This can be used to solve an equation g'(x) = ag(x). g=f(ax) is the solution.

Let’s say we want to solve the equation g'(x) = (a+b)g(x). Using the result above, possible solution is g_1(x)=f((a+b)x) and its derivative is g'_1(x) = (a+b)f((a+b)x) = (a+b)g_1(x).

But another solution could be g_2(x)=f(ax)f(bx), as (using the product rule) its derivative is g'_2(x) = (a+b)f(ax)f(bx) = (a+b)g_2(x).

As g_1(0) = f(0) = 1 and g_2(0) = f(0)f(0) = 1, both g_1 and g_2 go thru the point (0, 1) and has the same derivative. So they are the same function and f(ax+bx) = f(ax)f(bx). As the product of function calls is equivalent to summation of arguments, it follows that f(x) has properties of the exponential function a^x for some a = f(1) = 1 + 1 + \frac{1}{2} + \frac{1}{6} + \ldots = e (Euler’s number).

So e^x is a fixed point of differentiation.
